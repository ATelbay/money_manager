# 💰 Money Manager

Мобильное приложение для управления личными финансами на Android.

## О проекте

Money Manager помогает пользователям вести учёт доходов и расходов, управлять несколькими счетами в разных валютах и анализировать свои финансы. Приложение поддерживает импорт банковских выписок казахстанских банков с помощью AI.

## Возможности

- **Транзакции** — добавление, редактирование, поиск и фильтрация
- **Счета** — несколько счетов с агрегацией баланса
- **Категории** — раздельные категории доходов и расходов с иконками
- **Статистика** — столбчатые и круговые диаграммы, выбор периода
- **Импорт выписок** — PDF и фото для Kaspi, Freedom, Forte, Bereke, Eurasian (+ Gemini AI fallback)
- **Мультивалютность** — курсы Национального Банка Казахстана
- **Синхронизация** — хранение данных в Firebase Firestore
- **Аутентификация** — Google Sign-In через Firebase Auth
- **Настройки** — тема (System/Light/Dark), язык (RU/EN/KZ)

## Технологический стек

- **Язык:** Kotlin
- **UI:** Jetpack Compose + Material Design 3
- **Архитектура:** мультимодульная (36 Gradle-модулей)
- **Backend:** Firebase (Auth, Firestore, Crashlytics, Remote Config)
- **AI:** Google Gemini API
- **CI/CD:** GitHub Actions
